ERANEOS AI READINESS & MATURITY SCAN - DEMO INSTRUCTIONS

To test the application immediately:

METHOD 1: Direct File Opening (Recommended)
1. Right-click on "standalone-demo.html" in the file explorer
2. Select "Open with" → "Your web browser" (Chrome, Firefox, Edge, etc.)
3. The application will load directly in your browser

METHOD 2: File Explorer
1. Navigate to the folder in Windows Explorer
2. Double-click on "standalone-demo.html"
3. It should open in your default browser

METHOD 3: Browser Address Bar
1. Open your web browser
2. Press Ctrl+O (or File → Open)
3. Navigate to and select "standalone-demo.html"

WHAT YOU'LL SEE:
- Complete Eraneos-branded AI Maturity Assessment
- 7 dimensions with 3 questions each (simplified for demo)
- Interactive scoring with real-time maturity level updates
- Dashboard with radar and bar charts
- Professional UI matching Eraneos brand colors

FOR FULL PRODUCTION VERSION:
- Install Node.js from nodejs.org
- Run "npm install" in this folder
- Run "npm start" to launch development server
- Access at http://localhost:3000

The standalone demo works completely offline and demonstrates all core functionality except backend integrations (SharePoint, email).
